## How to use this in title formula

Per `lesson_title_seo_formula_2026_05.md`:

```
[Primary Keyword from above] [Curiosity Hook] | Sara and Eva [Sub-keyword]
```

Example:
- Primary: "Tooth Fairy" (121k volume)
- Curiosity: "and a Sneaky Dog"
- Sub: "Story for Kids"
- **Final title: "Tooth Fairy and a Sneaky Dog! 🦷 Sara and Eva Story for Kids"**

## Title vs description vs tag distribution

| Surface | Where keywords live |
|---|---|
| **Title** | 1-2 Primary Keywords at the start |
| **Description first 150 chars** | Primary Keyword + 2-3 Sub-keywords |
| **Description body** | Long-tail variations naturally |
| **Tags (15-25 per video)** | Mix of Primary + Sub + branded + competitor-adjacent (like nastya, vlad and niki, bluey, cocomelon) |
